    This is a Website for my students!

    In many of my advanced python classes this semester, my student's had the opportunity to build their own website! (and they are all beautiful websites!!!)
     However, hosting a website online is not something we usually do; the problem is that it costs money to host lots of websites, especially for the extended future. My students, however, were extremely excited about the idea of putting their website online. (And So was I!) It would be awesome to have it online, and remain online forever! This website is a solution to that problem! Through a service on the website pythonanywhere, I am able to host a Flask web app for free! But I am only able to host one website.

    So, as a way of having all my student's websites go online and remain online for free, I created this! A website that contains all of my students websites.
